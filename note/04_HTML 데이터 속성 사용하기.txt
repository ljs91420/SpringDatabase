# HTML 데이터 속성

	- 속성의 이름이 data-로 시작하는 모든 속성은 자바스크립트에서 쉽게 꺼내 쓸 수 있다.
	- 어트리뷰트에서 꺼낸 값을 자바스크립트로 전달할 때 아주 적합한 기능이다.
	- 타임리프를 사용한다면 th:data-xx를 통해 값을 전달할 수 있다.
	- data-로 시작하는 속성들은 자바스크립트에서 element.dataset 안에 들어있다.
	- 두 단어 이상으로 된 속성명을 사용하고 싶은 경우 HTML에서는 -를 사용해 잇고 자바스크립트에서는 camelCase로 접근해야 한다.
	
# 사용 예

	* HTML에서
	 
		<button class="sale-btn" data-fid="3">판매!</button>
		<button class="sale-btn" data-fruit-price="1500">판매!</button>
	
	* JS에서 
	
		console.log(saleButton.dataset.fid); // 결과 : 3
