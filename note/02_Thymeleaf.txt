# Thymeleaf

	- Spring Boot에서 밀어주는 뷰 템플릿 라이브러리
	- 기존의 JSP는 너무 많은 기능을 제공하기 위해 다소 불필요한 문법들이 포함되어 있었다.
	- JSP에서 기능을 많이 덜어내고 뷰를 만드는 데에만 집중한 템플릿 엔진
	- HTML의 형태를 많이 해치지 않고 최대한 형태를 유지하려는 특징을 보인다.
	
# Thymeleaf의 다양한 태그들

	- 기본적으로 HTML 속성 앞에 th:를 붙이는 방식으로 동작한다.
	- th:을 붙여서 속성을 적용하면 기존의 HTML 속성을 덮어쓴다. 
	
	- th:text - 해당 요소의 innerText를 채운다(태그 미적용)
	- th:utext - 해당 요소의 innerHTML을 채운다(태그가 적용됨, 보안 취약)
	
	- <form>, <input> 관련 태그들
	
		th:object : 자바빈 모델을 넘겨받으면 해당 폼에서 활용한다.
		th:field : th:object를 사용한 폼 내부에서 th:field를 사용해 변수를 매핑할 수 있다.(th:field를 사용해 input 태그의 name과 value 속성을 한 번에 채울 수 있다. 
		th:value : THML value 속성에 원하는 값을 채울 때 사용한다.
		
	- 자세한 내용은 th/form.html에 있음
